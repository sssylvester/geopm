Detailed Description

Factory object managing decider objects.

The DeciderFactory manages all instances of Decider objects. During construction the factory creates instances of all built in Decider classes as well as any Decider plugins present on the system. All Deciders then register themselves with the factory. The factory returns an appropriate Decider object when queried with a description string. The factory deletes all Decider objects on destruction.
